---
title: Input
---

The `Input` component is used to capture user input in a text field. It extends the HTML input element with additional props and supports various customization options.

<Preview name="Input" />

## Props

`Input` extends `InputHTMLAttributes<HTMLInputElement>` with following props:

<PropsSpec
  data={[
    {
      name: "fullWidth",
      type: "boolean",
      defaultValue: "false",
    },
    {
      name: "helperText",
      type: "string",
    },
    {
      name: "hasError",
      type: "boolean",
      defaultValue: "false",
    },
    {
      name: "label",
      type: "string",
    },
  ]}
/>
