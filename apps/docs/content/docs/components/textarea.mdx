---
title: TextArea
---

The `TextArea` component is used to capture multiline text input from the user. It extends the HTML textarea element with additional props and supports customization of the label.

<Preview name="TextArea" />

## Props

`TextArea` extends `InputHTMLAttributes<HTMLTextAreaElement>` with following props:

<PropsSpec
  data={[
    {
      name: "label",
      type: "string",
    },
  ]}
/>
