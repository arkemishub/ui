Install

How to install @arkejs/ui and its Tailwind CSS plugin?

  1. Install @arkejs/ui
    npm i @arkejs/ui
  2. Then add arkeUI plugin to your tailwind.config.js files:
    tailwind.config.js
     module.exports = {
        content: [
           './node_modules/@arkejs/{ui,table}/dist/**/*.{js,ts,jsx,tsx}',
        ],
        plugins: [require("@arkejs/ui/plugin")],
     }
